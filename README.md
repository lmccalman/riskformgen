# formgen
Engine to create static pages containing interactive forms.

## Planned features

- Uses Jinja2 templates with TailwindCSS and alpine.js for interactivity.
- Uses python dataclasses to define form declaritively, which are then rendered
  into static pages
- Tabbed interface with n sections, each with their own form questions.
- Supports conditional logic such as rendering a subsection with additional
  questions based on the answer tow question 1
- User can download a json with their form data at any time.
